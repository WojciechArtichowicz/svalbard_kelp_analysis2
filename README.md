# Analysis of the Svalbard kelp forest sea currents

### This repository was the data analysis and visualization part for paper [Seawater flow attenuation by structurally complex kelp forests in a high-latitude fjord](https://doi.org/10.1016/j.csr.2026.105646)

### Notebooks contain the following information:

1. [Sea current time series decomposition](time_series_decomposition.ipynb) - time series components visualization
2. [Speed / Velocity plots](velocity.ipynb) - velocity visualization
3. [Kelp analysis](kelp.ipynb) - kelp species analysis
4. [t-SNE analysis of velocity components](analysis_t-SNE.ipynb) - t-SNE analysis

Additional files:
1. [velocity_data_import](velocity_data_import.R)
2. [chemical_data_import](chemical_data_import.R)
3. [lib_import](lib_import.R)
4. [setup](setup.R)
